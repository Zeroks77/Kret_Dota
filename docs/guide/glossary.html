Glossary

Short, friendly definitions. Hover terms in the docs link here.

Report

A web page that summarizes matches for a period (month/patch) or a league.

Viewer

An interactive section inside a report (e.g., Players, Heroes, Wards) you can sort or filter.

Pre‑aggregation

Preparing data ahead of time so pages load fast. Like chopping vegetables before cooking.

Permalink

A link that remembers your current selections and filters.

OpenDota

Public data source for Dota 2 match and hero information.

League

A competition or tournament (e.g., The International) with its own report wrapper.
